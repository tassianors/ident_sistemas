\section{Aquisição dos dados para identificação do sistema}
\label{sec:get_data}
%===============================================================================

O sistema apresentado na seção (\ref{sec:modelling}) é a descrição matemática para
a planta do sistema, e a função de transferência utilizada é a que descreve a
posição angular do sistema com relação a tensão aplicada sobre os terminais do motor.

Na prática o que o que foi utilizado é um sistema como o apresentado na Figura (\ref{fig:sys_pid})
onde a planta que se quer identificar esta sendo controlada por um controlador do tipo PID
(Proporcional, integral diferencial) \cite{Ogata}. Onde aplica-se uma referencia e o sistema 
deve seguir esta referencia.

\begin{figure}[htbp]
	\center
	\includegraphics[width=0.9\columnwidth]{figures/sys_pid.eps}
	\caption{Representação do sistema utilizado para a coleta dos dados a serem utilizados no 
	processo de estimativa dos parâmetros da função $G(q)$.}
	\label{fig:sys_pid}
\end{figure}

Com este sistema foi possível fazer coleta dos dados dos sinais $r(t)$, $y(t)$ além do sinal
$u(t)$ que é o sinal de entrada da planta a ser identificada.

Com estes dados é possível proceder de duas maneiras distintas para estimar os valores dos 
parâmetros de $G(q)$, é possível apenas utilizar os sinais $r(t)$ e $y(t)$ e adicionar ao modelo
apresentado na seção (\ref{sec:modelling}) o modelo do controlador PID, já que este tem um modelo
conhecido e seus parâmetros foram previamente escolhidos (antes da simulação).

Outra maneira possível é utilizar apenas os sinais $u(t)$ e $y(t)$, desta forma pode-se ignorar a
existência do controlador PID, e utilizar o método desejado, apenas com o modelo do processo (\ref{eq:motor_pos_tf_disc}).

\subsection{Dados considerando-se o controlador PID}
%===============================================================================

Quando se considera o PID para a estimativa dos parâmetros, deve-se adicionar esta informação para
efetuar as estimativas. O modelo do controlador PID pode ser descrito como em (\ref{eq:pid}). 

\begin{equation}
u(t)=k_p(r(t)-y(t))+k_i \int_{0}^{t}r(\tau)-y(\tau)d\tau + k_d(\frac{\mathrm{d} r(t)}{\mathrm{d} t}-\frac{\mathrm{d} y(t)}{\mathrm{d} t})
\label{eq:pid}
\end{equation}

Nas medidas efetuadas em laboratório foi utilizado os seguintes valores para o ajuste do controlador
PID, Tabela (\ref{tab:pid}).

\begin{table}[htbp]
  \begin{center}
	\caption{Valores dos parâmetros para ajuste do controlador PID utilizado}
	\label{tab:pid}
	\begin{small}
	  \begin{tabular}{cl}
		\hline
		Parâmetro	& valor  \\
		\hline
		kp		& 3      \\
		ki		& 0      \\
		kd		& 0.05   \\
		\hline
	  \end{tabular}
	\end{small}
  \end{center}
\end{table}


Aplicando-se os valores da Tabela (\ref{tab:pid}) sobre a equação (\ref{eq:pid}) obtém-se
o controlador PID utilizado:

\begin{equation}
u(t)=3(r(t)-y(t))+0.05(\frac{\mathrm{d} r(t)}{\mathrm{d} t}-\frac{\mathrm{d} y(t)}{\mathrm{d} t})
\nonumber
\end{equation}

Utilizando-se a transformada de Laplace tem-se:

\begin{equation}
PID(s)=k_p+k_d\;s
\nonumber
\end{equation}

Utilizando a aproximação de Euler Forward (\ref{eq:modelling_forward}) obtém-se:

\begin{equation}
PID(z)=k_p+k_d\;(z-1)/T_s=3+5z-5=5z-2
\label{eq:pid_disc}
\end{equation}
