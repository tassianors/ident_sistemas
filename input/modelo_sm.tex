\section{Modelo Completo}
\label{sec:non_arx}
%===============================================================================

Como apresentado na seção (\ref{sec:arx}) o modelo ARX não consegue representar o
sistema (\ref{eq:system}) completamente, e a estimativa dos parâmetros da Função
de transferência são polarizados. Para contornar este problema utilizaremos um
modelo para descrever o sistema (\ref{eq:system}) de forma completa. \cite{system_identification}

O modelo escolhido para representar o sistema real é apresentado em (\ref{eq:compl}).

\begin{equation}
G(q, \theta )=\frac{a}{q-b}\;\;\;\;\;H(q, \theta)=\frac{q-c}{q-d}
\label{eq:compl}
\end{equation}

Utilizando o estimador ótimo (\ref{eq:estimador}) obtém-se a equação de diferenças apresentada
em (\ref{eq:gmq_eq}). Utilizando o {\it{script}} \ref{anx_gmq_simul} obtém-se o resultado
para os parâmetros $a$, $b$, $c$ e $d$ para a função $G(q, \theta)$ e $H(q, \theta)$.

\begin{equation}
\hat{y}(t/t-1, \theta)=H^{-1}(q, \theta)G(q, \theta)u(t)+[1-H^{-1}(q, \theta)]y(t)
\label{eq:estimador}
\end{equation}

\begin{equation}
\begin{matrix}
\hat{y}(t/t-1, \theta)= & a\;u(t-1)-ad\;u(t-2) +(d-c)y(t-1)\\
 & -b(d-c)y(t-2)+(b+c)\hat{y}(t-1/t-2, \theta)\\
 & -cb\;\hat{y}(t-2/t-3, \theta)
\end{matrix}
\label{eq:gmq_eq}
\end{equation}

Desta forma temos para o método dos mínimos quadrados as seguintes equiões:

\begin{equation}
\Theta = \begin{bmatrix}
a\\ 
a\;d\\ 
(d-c)\\ 
b(d-c)\\ 
(b+c)\\ 
c\;b
\end{bmatrix}
\nonumber
\end{equation}

\begin{equation}
\varphi (t)=\begin{bmatrix}
u(t-1)\\ 
-u(t-2)\\ 
y(t-1)\\ 
-y(t-2)\\ 
\hat{y}(t-1/t-2,\theta)\\ 
-\hat{y}(t-2/t-3,\theta)
\end{bmatrix}^T
\nonumber
\end{equation}

das funções de transferência $G(q)$ e $H(q)$ respectivamente.
