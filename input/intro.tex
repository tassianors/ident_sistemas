\section{Introdução}

Neste trabalho será apresentado um modelo ARX para um sistema que não pode ser
completamente representado por este modelo (\ref{eq:system}). Em seguida será
utilizado um modelo mais completo e será feito um comparativo qualitativo das
estimativas obtidas para o modelo utilizando cada um dos métodos.

\begin{equation}
G_0(q)=\frac{2}{q-0.8}\;\;\;\;\;H_0(q)=\frac{q-0.9}{q-0.5}
\label{eq:system}
\end{equation}
